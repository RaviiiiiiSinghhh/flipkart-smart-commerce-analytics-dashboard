# Flipkart Smart Commerce Analytics Dashboard

# 📊 Project Overview

This Power BI dashboard analyzes Flipkart sales performance, profitability, customer ratings, payment preferences, and regional trends through interactive visualizations.

- Power BI
- DAX
- Microsoft Excel

# 📈 Key Insights

- Total Sales: ₹75.21M
- Total Profit: ₹15.0M
- Electronics category generated the highest sales.
- Wallet is the most preferred payment method.
- North region has the highest sales.
